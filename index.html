About the Visualization

This, CS 498 Data Visualization, d3.js final project uses the hybrid approach of an Interactive Slideshow in order to narrate the visual story of Financial Market dataset (FINVIZ.COM). The interactive slideshow takes the user through the Dataset of the July 09 2018 Single Trading Day (mode details to follow).

The Interactive Slideshow is built on scenes with annotations having parameters connected with triggers and constantly follows a template for visual consistency. Specifics with regards all these elements will be detailed later on in the presentation.

Recommended Settings: As the visualizations use a preset static canvas of 1024x768 pixels, the viewing size of the Interactive Slideshow is at least 1280x800 pixels in order to accommodate the surrounding text and scene elements.(FINVIZ.COM). ELEMENTS OF NARRATION

  1. Scenes

    2. The slides that make up the presentation; these slides can be navigated through guiding arrows in the corner. The use of CSS and FullPage.js allows for a consistent template look for the Interactive Slideshow visualization; each page is transitioned using the same method, while the elements are kept the same - both font size and family.
    Additionally, there is a page navigation control (on the right bottom side) which allows users to navigate between different slides. User can press Esc key at any point of the Interactive Slideshow to get an overview of all scenes.

  2. Annotations

    3. Various tooltips, labels & legends; that display key attributes from the Dataset - such as Company, Country, Market Capitalization and Sector. The tooltips are triggered when a user interactively moves the mouse over the visualization elements and this invokes the display parameters for the each relevant item.

    In general, the annotations that are being used in the visualizations all have a similar process of using triggers to change the hidden parameter of the annotation. For example, the INTERACTIVE ANALSYSIS OF THE WORLD chart has an initial state of the hidden parameter which controls the display of the annotations set to false. As a user triggers changes to the chart parameters, it also changes this hidden parameter to true. In turn the annotations disappear until the user resets chart to default - thus zooming out back to the default visualization parameters.

  3. Parameters

    4. The main parameters of this story are Company, Sector, Market Capitalization and Country. There are additional parameters that are listed in the Dataset, yet those are minor.

  4. Triggers

    5. Zoom and Brush filter, buttons with affordances are used in order to trigger updates in visuals based on new parameters.

    Both parameters and triggers are intensively used in all of the visualizations, with an emphasis on the INTERACTIVE ANALSYSIS OF THE WORLD and MAJOR COMPANIES INTERACTIVE INSIGHTS charts. For example – in INTERACTIVE ANALSYSIS OF THE WORLD parameters for the (x, y) coordinates of each Bubble are set to an initial central position for the All Companies Visualization. As the user selects various grouping options (Sector, Country and Market Capitalization) it triggers the change of the Bubble’s (x, y) coordinates parameter to their respective groupings. In the MAJOR COMPANIES INTERACTIVE INSIGHTS Sunburst Visualization, every mouse click on a region is a trigger for the path and arc parameters. By clicking within a region, you can zoom into the data to take a look at the information underneath. Clicking the central circle will trigger the parameters to return to its values one up in the hierarchy.

THE DATASET

    With approximately 10,000 companies, The FINVIZ.COM is considered as one of major financial data providers, it searches through large amount of stock data and collects them in a single source.

    Relevant stock exchanges (where the company is listed) - all stocks listed on: National Association of Securities Dealers Automated Quotation (NASDAQ), New York Stock Exchange (NYSE) and American Stock Exchange (AMEX) are available.

    Snapshot data: July 09, 2018 (Single Day performance). The data was slightly cleaned and pre-processed using common data curation activities

  1. Sector

    2. Companies are divided into several groups - sectors - according to their business activities.

  2. Industry

    3. Companies in a common sector are further divided by products and services into smaller groups - industries.

  3. Country

    4. The geographic location of a company (listed on U.S. markets).

  4. Market Capitalization (Cap)

    5. The total dollar value of all of a company's outstanding shares. Market capitalization is a measure of corporate size.

  5. P/E

    6. A popular valuation ratio of a company's current share price compared to its per-share earnings (trailing twelve months). Low P/E value indicates a stock is relatively cheap compared to its earnings.

  6. Price

    7. The current stock price or the close price during the last trading session.

  7. Change

    8. The difference, in %, between current close and previous close price.

  8. Volume

    9. Total number of shares traded for a given stock today, or during the last trading session.
